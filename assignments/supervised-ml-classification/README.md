# 📘 Assignment: Supervised Machine Learning — Classification

## 🎯 Objective

Students will build, evaluate, and interpret a binary classification model using Python and scikit-learn. The assignment covers data loading, feature engineering, cross-validation, evaluation metrics (precision, recall, ROC), and basic model interpretation.

## 📝 Tasks

### 🛠️ Data exploration & preprocessing

#### Description
Load the provided dataset, inspect it for missing values and distributions, and perform basic preprocessing (imputation, scaling, encoding if needed).

#### Requirements
Completed work should:

- Load `sample_data.csv` and display summary statistics
- Handle missing values appropriately
- Create at least one new feature (feature engineering)

### 🛠️ Model training & evaluation

#### Description
Train a supervised classifier (e.g., Logistic Regression, Random Forest) using cross-validation. Tune at least one hyperparameter and evaluate using appropriate metrics.

#### Requirements
Completed work should:

- Split data using cross-validation (or train/test) and report results
- Show precision, recall, F1-score, and ROC AUC
- Provide a confusion matrix and short discussion of errors

### 🛠️ Interpretation & report

#### Description
Interpret your model's behavior and write a short report (1–2 paragraphs) explaining feature importance and what could be improved.

#### Requirements
Completed work should:

- Provide feature importances or coefficients
- Recommend two possible next steps to improve model performance
